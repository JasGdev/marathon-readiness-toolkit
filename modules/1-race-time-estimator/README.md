# Race Time Estimator

## What this tool does
Estimates a realistic marathon time range based on a recent 5K, 10K, or Half Marathon effort.
This tool focuses on expectation-setting, not prediction guarantees.

## What it is NOT
- Not a training plan
- Not coaching advice
- Not a performance promise

## Inputs
- Recent race distance (5K / 10K / Half)
- Finish time

## Outputs
- Marathon time range
- Contextual explanation of uncertainty

## Intended use
- Early and mid training cycle
- Reality check for goal setting
- Revisit after tune-up races
